# python-introduccion

Este repositorio tiene 2 partes:

* La parte 1 contiene una introducción (no exhaustiva) a Python. 
* La parte 2 contiene una introducción a NumPy.

Para cada parte se tiene tanto el archivo de Jupyter Notebook (.ipynb) como el html generado a partir de dicho archivo. En ambos casos también se tiene los archivos con las respectivas salidas (resultados de la ejecución).
